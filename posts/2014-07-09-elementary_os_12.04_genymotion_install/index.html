GenyMotionのelementaryOS12.04へのInstall

爆速Android用のエミュレータ GenyMotion導入

最近巷で爆速と噂のGenyMotion入れてみます。
※ Ubuntuでも同じ導入手順です。

VirtualBoxのInstall

UbuntuソフトウェアセンターでVirtualboxをインストールします。

GenyMotionのDownload


AndroVMのダウンロードサイトにてダウンロード。
ダウンロード前にここでユーザ登録を行う必要があります。

GenyMotionのInstall

ダウンロード後のファイルの配置

お好きな場所に配置します。私は/opt/に配置しました。

cd ~/Downloads/
sudo mv genymotion{*version番号*}.bin /opt/

実行権限の変更

cd /opt/
sudo chmod 755 genymotion{*version番号*}.bin

installerの実行

sudo ./genymotion{*version番号*}.bin

この例では /opt/genymotion/ にインストールされます。

installerの削除

インストーラーは不要なので削除します

sudo rm genymotion{*version番号}.bin

pathを通す

このままでは/opt/genymotion以下からしか呼び出せません。
pathを通します。

echo 'export PATH=$PATH:/opt/genymotion' >> ~/.bashrc
source ~/.bashrc

GenyMotion実行

terminalでgenymotionで実行

User情報の追加

登録したユーザ情報を入力する必要があります。

デバイスの追加

Addボタンからデバイスを追加します。

一覧からデバイスイメージを選択します。

デバイス名を登録します。

デバイスの起動

Playボタンからデバイスを起動します。

これで起動完了です。

快速快適GenyMotion!めっちゃサクサクですよ！

また、Terminalからではなく、Eclipse or AndroidStudioプラグインをインストールして そこから立ち上げることも可能です。 (むしろソッチのほうがおおいかな)
