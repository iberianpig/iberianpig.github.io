Vimの折りたたみをしない設定

いつも+++とかrspecをvim上で表示したり、markdownでブログ書いたりとかしてる時に 毎回折りたたみ開くのが面倒だったので、いっそ折りたたみしないことにした。

set foldmethod=syntax
let perl_fold=1
set foldlevel=100
